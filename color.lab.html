Color Mixing Lab

Choose an object color and a light color to see the resulting perceived color.

Select an **Object Color** and a **Light Color** above.

The **Resulting Color** will be shown in the box above. For example:

  • If a **Red** object is illuminated by **Green** light, it appears **Black** (as the object absorbs green light).
  • If a **Tomato (Red)** is under **Blue** light, it appears **Black**.
  • If a **Green** leaf is under **Red** light, it appears **Black**.
  • If a **White** object is under **Red** light, it appears **Red**.
  • If a **Yellow** object (Red + Green pigments) is under **Blue** light, it appears **Black**.
